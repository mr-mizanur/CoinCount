# CoinCount 

CoinCount is a simple and efficient Expense Tracker application built with **Next.js 15+**, **TypeScript** and **MongoDB**. It helps you manage your daily expenses and keep track of your financial habits in real-time.

##  Features
- **Add Expenses:** Log your daily spending with titles, amounts, and categories.
- **Track Expenses:** View a list of your recent transactions.
- **Auto Calculation:** Automatically calculates the total expenditure.
- **Database Integrated:** Uses MongoDB for persistent data storage.

##  Tech Stack
- **Framework:** Next.js (App Router)
- **Database:** MongoDB (Native Driver)
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **Language:** TypeScript
